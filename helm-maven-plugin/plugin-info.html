Plugin Documentation

This report describes goals, parameters details, requirements and sample usage of this plugin.

Goals

Goals available for this plugin:

Goal Description
helm:dependency-update This goal downloads the helm dependencies configured to the charts directory.
helm:deploy-helm This goal deploys the generated helm package to a remote repository.
helm:download-helm This goal downloads the helm executable specified.
helm:generate-helm-sources This goal generates the helm sources.
helm:help Display help information on helm-maven-plugin.
Call mvn helm:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
helm:package This goal packages the helm project. It does so by simply calling helm package on the sources generated.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>de.eitco.cicd.helm</groupId>
          <artifactId>helm-maven-plugin</artifactId>
          <version>4.0.1</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>de.eitco.cicd.helm</groupId>
        <artifactId>helm-maven-plugin</artifactId>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

build-utilities:write-file

Full name:

de.eitco.cicd:build-utilities-maven-plugin:0.0.7-SNAPSHOT:write-file

Description:

This goal generates a text file.

Attributes:

  • The goal is not marked as thread-safe and thus does not support parallel builds.

Required Parameters

Name Type Since Description
<content> String - The content to write to the file.
User Property: file.content
<targetFile> File - The name of the file to generate.
User Property: target.file

Optional Parameters

Name Type Since Description
<append> boolean - Whether content should be appended to the file - or the file should be overridden.
Default: false
User Property: append.file
<skip> boolean - Whether the execution should be skipped.
Default: false
User Property: write.file.skip

Parameter Details

build-utilities:dependencies-version-properties

Full name:

de.eitco.cicd:build-utilities-maven-plugin:0.0.7-SNAPSHOT:dependencies-version-properties

Description:

This goal generates properties, containing the version and file system path of every direct dependency the current project has. For every dependency, two properties are generated: 'maven.dependency.<dependency-conflict-id>.file' set to the absolute file system path of the dependency and 'maven.dependency.<dependency-conflict-id>.version' set to the version of the dependency. The dependency conflict id is constructed as follows: <groupId>:<artifactId>:<type>[<classifier>]

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: initialize.

Optional Parameters

Name Type Since Description
<addFiles> boolean - This parameter specifies whether to add properties holding the files of the dependencies.
Default: false
User Property: dependencies-version-properties.add-files
<skip> boolean - If this parameter is set to true the plugin execution is skipped completely.
Default: false
User Property: dependencies-version-properties.skip

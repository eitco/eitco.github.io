build-utilities:set-artifact

Full name:

de.eitco.cicd:build-utilities-maven-plugin:0.0.7-SNAPSHOT:set-artifact

Description:

This goal sets the artifact file of the current project. This is intended for highly customized builds where maven is unable to deduce the builds main artifact by itself.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is not marked as thread-safe and thus does not support parallel builds.

Required Parameters

Name Type Since Description
<file> File - The file to attach to the project.
Default: ${project.build.directory}/${project.artifactId}.${project.packaging}
User Property: artifact.file

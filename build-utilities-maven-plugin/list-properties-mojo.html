build-utilities:list-properties

Full name:

de.eitco.cicd:build-utilities-maven-plugin:0.0.7-SNAPSHOT:list-properties

Description:

This goal prints all properties and their value currently available. This is mostly thought of for debugging purposes.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: initialize.

Optional Parameters

Name Type Since Description
<outputFile> File - This parameter sets the file to write the properties to. If not set the properties will simply be logged.
<prefix> String - This parameter specifies a prefix. If set, only properties beginning with this prefix will be listed.

arveo - Content Services Platform

arveo is a multi-client and 100% cloud-ready content services platform.

EQL (EITCO Query Language)

The EITCO query language provides a unified interface for searches.

Audit Service

The Audit Service for database auditing.

Document Conversion Service

The EITCO Document Conversion Service is a microservice capable of converting different document formats and extracting full text of a document.

Job Service

Here you can find documentation about the EITCO Job Service.

Integration Service

The Enterprise Integration Service is a Spring Service based on the Apache Camel framework.

Universal Import Service

The Universal Import Service is used to import data into the arveo content repository service.

Cloud Services

The Cloud Services offer functionality like centralized configuration and service registration.
